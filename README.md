<details>
<summary><strong>👀 Para Rodar Localmente</strong></summary><br />

1. Clone o repositório   
  `git@github.com:leilaMoraes/find_cep.git`
2. Navegue até a pasta do repositório clonado  
    `cd find_cep`
3. Para rodar a aplicação    
  `flutter run`
</details>
